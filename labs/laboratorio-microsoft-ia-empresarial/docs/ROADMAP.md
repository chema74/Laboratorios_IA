# Roadmap

## V0 documental
- Definición de visión, arquitectura conceptual, límites y catálogo de proyectos.
- Creación de plantillas estándar para documentación técnica.

## V1 local funcional
- Implementaciones locales simuladas sin APIs reales obligatorias.
- Ejecución con datos sintéticos y trazabilidad básica.

## V2 opcional con Microsoft Graph API
- Integración real opcional mediante variables .env.
- Mantenimiento de fallback local como requisito de continuidad.

## V3 futura extendida
- Posible extensión documentada con Microsoft 365 real, Copilot, Power Platform o Azure.
- Activación solo cuando sea necesaria y con controles de seguridad reforzados.

## Fuera de alcance
- Producción con datos reales.
- Automatizaciones críticas de negocio en vivo.
- Dependencia obligatoria de servicios de pago.
- Promesas de SLA, coste o rendimiento empresarial.
