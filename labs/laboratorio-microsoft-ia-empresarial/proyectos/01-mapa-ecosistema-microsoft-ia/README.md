# 01 - Mapa Ecosistema Microsoft IA

## Objetivo
Definir y generar localmente un mapa tecnico del ecosistema Microsoft orientado a IA empresarial usando componentes sinteticos.

## Estado actual
V1 local funcional minima implementada.

## Entradas
- `datos_ejemplo/ecosistema_microsoft_ia_sintetico.json`
- `datos_ejemplo/configuracion_mapa_microsoft_ia.json`

## Salidas
- Informe Markdown del mapa ecosistemico.
- JSON consolidado con clasificaciones por categoria y capa.

## Ejecucion
`python .\proyectos\01-mapa-ecosistema-microsoft-ia\src\generador_mapa_ecosistema.py --ecosystem .\proyectos\01-mapa-ecosistema-microsoft-ia\datos_ejemplo\ecosistema_microsoft_ia_sintetico.json --config .\proyectos\01-mapa-ecosistema-microsoft-ia\datos_ejemplo\configuracion_mapa_microsoft_ia.json --output-md .\proyectos\01-mapa-ecosistema-microsoft-ia\informes\informe_mapa_ecosistema_microsoft_ia.md --output-json .\proyectos\01-mapa-ecosistema-microsoft-ia\informes\mapa_ecosistema_microsoft_ia.json`

## Limites
Sin Microsoft 365 real, sin OAuth real, sin Microsoft Graph API real, sin Azure obligatorio y sin IA real.
