# 09 - Informe Microsoft IA Empresarial

## Objetivo
Generar un informe consolidado del laboratorio Microsoft IA empresarial en modo local y sintético.

## Estado actual
V1 local funcional mínima implementada.

## Entradas
- `datos_ejemplo/resultados_microsoft_ia_empresarial.json`
- `datos_ejemplo/configuracion_informe_microsoft_ia.json`

## Salidas
- Informe Markdown consolidado.
- JSON de evaluación empresarial simulada.

## Ejecución
`python .\proyectos\09-informe-microsoft-ia-empresarial\src\generador_informe_microsoft_ia.py --results .\proyectos\09-informe-microsoft-ia-empresarial\datos_ejemplo\resultados_microsoft_ia_empresarial.json --config .\proyectos\09-informe-microsoft-ia-empresarial\datos_ejemplo\configuracion_informe_microsoft_ia.json --output-md .\proyectos\09-informe-microsoft-ia-empresarial\informes\informe_microsoft_ia_empresarial.md --output-json .\proyectos\09-informe-microsoft-ia-empresarial\informes\informe_microsoft_ia_empresarial.json`

## Límites
Sin Microsoft real, sin Microsoft Graph API real, sin OAuth real, sin Azure obligatorio, sin IA real y sin datos reales.
