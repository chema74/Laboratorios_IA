# Guion Demo Microsoft IA Empresarial

## Resumen ejecutivo
Demostración local de Laboratorio Microsoft IA Empresarial Simulado en cadena completa de 10 módulos simulados.

## Narrativa empresarial de uso
PYME ficticia que coordina ventas, operaciones y soporte con flujos Microsoft simulados.

## Recorrido por módulos
- 01-mapa-ecosistema-microsoft-ia
- 02-simulador-outlook-ia-empresarial
- 03-simulador-onedrive-word-ia
- 04-simulador-excel-ia-analitica
- 05-simulador-teams-tareas-ia
- 06-conector-copilot-fallback-local
- 07-gobierno-permisos-microsoft-simulado
- 08-trazabilidad-automatizaciones-microsoft
- 09-informe-microsoft-ia-empresarial
- 10-demo-microsoft-ia-empresarial-completa

## Cadena completa
mapa del ecosistema -> Outlook simulado -> OneDrive/Word simulados -> Excel analítico -> Teams/tareas -> Copilot fallback local -> gobierno de permisos -> trazabilidad -> informe empresarial -> demo final.

## Evidencias simuladas
- Informes markdown por módulo.
- Salidas JSON reproducibles.
- Registros locales de ejecución.
