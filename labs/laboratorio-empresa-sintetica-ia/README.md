# Laboratorio de Empresa Sintética para IA Empresarial

**Subtítulo:** Entorno local gratuito para generar datos, documentos, eventos y escenarios ficticios que permitan probar sistemas empresariales sin usar datos reales.

## Propósito del repositorio
Este repositorio implementa un laboratorio técnico local para construir y operar una empresa ficticia de extremo a extremo. Su objetivo es generar artefactos sintéticos reutilizables para evaluación técnica, diseño de procesos y demostraciones de portfolio.

## Problema que resuelve
Permite demostrar capacidades de arquitectura, trazabilidad y evaluación sin exponer información sensible de clientes o entornos productivos.

## Estado actual
- V1 local integradora completada.
- 10 proyectos internos implementados.
- 70 tests pasando (`python -m pytest .\proyectos -q`).
- Demo narrativa final disponible.
- Sin datos reales.
- Sin APIs externas obligatorias.
- Sin claves.
- Sin servicios de pago obligatorios.
- Sin IA funcional real todavía.
- Sin sistema productivo.

## Qué contiene hoy
- Generación de empresa sintética base.
- Simulación de eventos operativos.
- Fábrica de documentos sintéticos.
- Escenarios de prueba para agentes simulados.
- Simulación de crisis y revisión humana sintética.
- Gemelo digital operativo ligero sintético.
- Laboratorio de privacidad de datos sintéticos.
- Comparador de flujos manual/automatizado/agente simulado.
- Demo narrativa semanal completa que integra proyectos 01 a 10.

## Comandos principales
```powershell
python -m pytest .\proyectos -q
python .\proyectos\10-demo-narrativa-empresa-completa\ejecutar_demo.py --seed 42 --dias 7
```

## Qué no contiene
- No hay IA (Artificial Intelligence – Inteligencia Artificial) real en ejecución.
- No hay API (Application Programming Interface – Interfaz de Programación de Aplicaciones) externa ni productiva.
- No hay datos reales ni cliente real.
- No hay agentes reales en ejecución.
- No hay sistema productivo.
- No hay integración real con Google Workspace o Microsoft 365.

## Uso en el portfolio técnico
El repositorio alimenta áreas futuras de evaluación técnica: Big Data empresarial, evaluación de agentes, seguridad defensiva y demos empresariales integradas para contexto PYME (Small and Medium-sized Enterprise – Pequeña y Mediana Empresa).
