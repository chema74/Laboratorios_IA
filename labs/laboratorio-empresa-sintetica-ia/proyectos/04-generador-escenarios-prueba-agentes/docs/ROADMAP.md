# Roadmap de 04-generador-escenarios-prueba-agentes

## Fase documental
- Definición de alcance y criterios de salida.

## V1 (Version 1 - Versión 1) local mínima (implementada)
- Carga de empresa, eventos y documentos desde JSON.
- Fallback interno para ejecución y tests autónomos.
- Generación determinista de escenarios por tipo.
- Uso de catálogos controlados para acción recomendada y dificultad.
- Exportación JSON, CSV, resumen y Markdown por escenario.
- CLI local y pruebas unitarias base.

## Fase de ampliación local
- Mayor trazabilidad entre escenario y artefactos de origen.
- Reglas más finas de riesgo y privacidad por dominio.
- Cobertura extendida de criterios de evaluación.

## V2 (Version 2 - Versión 2) profesional futura
- Integración opcional con sistemas de evaluación operativa.
- Métricas comparativas y anotación humana avanzada.
- Extensión para flujos empresariales de PYME (Small and Medium-sized Enterprise - Pequeña y Mediana Empresa).

## Límites vigentes
- Sin ejecución de agentes reales.
- Sin IA (Artificial Intelligence - Inteligencia Artificial) real.
- Sin API (Application Programming Interface - Interfaz de Programación de Aplicaciones) externa activa.
- Sin datos reales.
- Sin servicios de pago.
