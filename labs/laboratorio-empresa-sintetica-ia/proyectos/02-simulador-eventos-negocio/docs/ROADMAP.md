# Roadmap de 02-simulador-eventos-negocio

## Fase documental
- Alcance y límites del simulador definidos.

## V1 (Version 1 – Versión 1) local mínima (implementada)
- Carga de empresa sintética desde JSON.
- Fallback interno para ejecución autónoma en tests.
- Simulación determinista de eventos operativos por día.
- Exportación JSON/CSV y resumen de simulación.
- CLI local y pruebas unitarias iniciales.

## Fase de ampliación local
- Reglas de correlación entre eventos (ej. pedido -> pago -> incidencia).
- Mayor realismo temporal de estados y severidades.
- Escenarios de estrés y anomalías controladas.

## V2 (Version 2 – Versión 2) profesional futura
- Integraciones externas opcionales y desacopladas.
- Enlaces con módulos de evaluación y seguridad defensiva para IA (Artificial Intelligence – Inteligencia Artificial).
- Evolución para ecosistemas API (Application Programming Interface – Interfaz de Programación de Aplicaciones) empresariales.

## Límites vigentes
- Sin API externa activa.
- Sin datos reales.
- Sin servicios de pago.
- Sin proveedores LLM en esta fase.
