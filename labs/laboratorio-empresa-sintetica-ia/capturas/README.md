# Capturas V1

## Propósito de esta carpeta
La carpeta `capturas/` queda preparada para almacenar capturas de apoyo visual del repositorio `laboratorio-empresa-sintetica-ia` en su estado V1 local integradora.

## Estado actual
Actualmente no se incluyen capturas si no existen archivos de imagen en esta carpeta.

## Política de incorporación
- Las capturas se añadirán manualmente.
- Cada captura debe revisarse antes de subirse al repositorio.
- No deben incluir datos reales.
- No deben incluir claves, tokens o credenciales.
- No deben incluir rutas privadas sensibles ni información personal.
- Deben mostrar solo datos sintéticos y salidas locales del laboratorio.

## Documentos asociados
- `PLAN_CAPTURAS_V1.md`: listado recomendado de capturas para documentar la V1.
- `GUIA_CAPTURAS_PORTFOLIO.md`: criterios y procedimiento para capturas útiles de portfolio técnico.
