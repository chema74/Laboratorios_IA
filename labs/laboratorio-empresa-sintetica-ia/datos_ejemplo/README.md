# datos_ejemplo

Este directorio reúne salidas de datos sintéticos generadas localmente por los módulos del laboratorio.

## Estado actual
- Contiene artefactos de la V1 local integradora.
- Todos los datos son sintéticos.
- La generación es reproducible en entorno local.
- No contiene datos reales ni información de cliente real.

## Uso recomendado
- Revisar resultados por subcarpeta según módulo.
- Interpretar el contenido como evidencia técnica de laboratorio, no como operación productiva.
