# Roadmap del Laboratorio

## Estado de fases V1
- Fase 0 documental: completada.
- Fase 1 generador base: completada.
- Fase 2 eventos y documentos: completada.
- Fase 3 escenarios y crisis: completada.
- Fase 4 revisión humana, privacidad y comparador: completada.
- Fase 5 demo narrativa completa: completada.

## V2 futura (pendiente)
- API local con FastAPI para orquestación.
- Dashboard local en HTML o Streamlit.
- Persistencia analítica con DuckDB y Parquet.
- Exportación de paquete de evidencias.
- Integración opcional con APIs gratuitas mediante `.env`.
- Extensión documentada futura para Google o Microsoft, sin dependencia obligatoria.

## Nota de alcance
La V1 actual se mantiene como laboratorio local, gratuito, reproducible y sintético para IA empresarial. No es un sistema productivo.
