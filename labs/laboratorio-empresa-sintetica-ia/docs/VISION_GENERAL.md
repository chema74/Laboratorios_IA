# Visión General

El Laboratorio de Empresa Sintética para IA Empresarial es un entorno local, gratuito, reproducible y sintético orientado a portfolio técnico. Su finalidad es diseñar y validar capacidades de IA empresarial sin depender de datos reales.

## Visión estratégica
- Crear una empresa ficticia integral con procesos, eventos y documentos simulados.
- Proveer insumos consistentes para pruebas de agentes, analítica y decisiones.
- Permitir evaluación técnica con evidencia rastreable y reutilizable.

## Papel dentro del portfolio
Este repositorio funciona como proveedor transversal para iniciativas futuras de analítica, automatización orientada a API (Application Programming Interface - Interfaz de Programación de Aplicaciones), seguridad y evaluación operativa.

## Estado actual de la V1
- V1 local integradora completada.
- 10 proyectos internos implementados.
- 70 tests pasando.
- Demo narrativa final disponible.
- Sin datos reales ni claves.
- Sin APIs externas obligatorias ni servicios de pago.
- Sin IA funcional real todavía.
- Sin sistema productivo.
