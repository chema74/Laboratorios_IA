# Principios Free-First

## Objetivo
Priorizar un entorno local, gratuito y auditable para construir capacidades progresivas.

## Modo 1 - Local sin API
- Operación completamente local.
- Sin dependencia de red ni proveedores externos.
- Base recomendada para la V1 (Version 1 - Versión 1).

## Modo 2 - API gratuita opcional mediante .env
- Integración opcional con API (Application Programming Interface - Interfaz de Programación de Aplicaciones) gratuitas.
- Activación controlada por variables de entorno.
- Uso estrictamente no obligatorio para mantener el enfoque local-first.

## Modo 3 - API de pago documentada como V2 futura
- Servicios comerciales solo como evolución V2 (Version 2 - Versión 2).
- Sin dependencia contractual en la fase actual.
- Evaluación económica y operativa diferida.

## Principio rector
Toda capacidad de IA (Artificial Intelligence - Inteligencia Artificial) debe poder degradarse a modo local para mantener continuidad técnica en una PYME (Small and Medium-sized Enterprise - Pequeña y Mediana Empresa).
