# Límites y Fuera de Alcance

## Límites de esta V1
- No existe IA (Artificial Intelligence – Inteligencia Artificial) funcional en producción.
- No existe API (Application Programming Interface – Interfaz de Programación de Aplicaciones) productiva.
- No existe dashboard productivo.
- No se usan datos reales.
- No existe cliente real.
- No hay integraciones reales con sistemas externos.

## Fuera de alcance explícito
- Integración real con Google Workspace.
- Integración real con Microsoft 365.
- Auditoría legal real.
- Cumplimiento regulatorio real certificado.
- Predicción empresarial real.
- Benchmark real de desempeño empresarial.
- Ahorro real garantizado.
- Automatización empresarial real en producción.
- Recomendación empresarial real.
- Claves reales o secretos operativos.

## Alcance permitido
- Diseño documental y técnico de laboratorio.
- Estructura modular reproducible.
- Evidencias sintéticas para revisión de portfolio.
- Definición de evolución V2 futura sin compromiso productivo.
