# Evidencias V1

## Propósito de esta carpeta
La carpeta `evidencias/` reúne documentación para revisar, de forma técnica y ordenada, los artefactos generados por la V1 local integradora del repositorio `laboratorio-empresa-sintetica-ia`.

## Para qué sirve
- Centraliza el contexto de revisión de resultados generados en local.
- Facilita el recorrido de evaluación para portfolio técnico.
- Reduce ambigüedad sobre alcance, límites y valor de cada salida.

## Alcance de las evidencias
- Las evidencias proceden de datos sintéticos generados dentro del repositorio.
- No incluyen datos reales.
- No representan actividad de cliente real.
- No corresponden a un sistema productivo.
- Se orientan a revisión técnica, trazabilidad y comunicación de arquitectura.

## Documentos de apoyo en esta carpeta
- `INDICE_ARTEFACTOS_V1.md`: inventario de artefactos por proyecto y guía de lectura.
- `GUIA_REVISION_TECNICA.md`: secuencia recomendada de validación y criterios de interpretación.

## Uso recomendado
1. Ejecutar validación técnica global.
2. Revisar artefactos por módulo en orden funcional.
3. Contrastar resultados con límites declarados del laboratorio.
