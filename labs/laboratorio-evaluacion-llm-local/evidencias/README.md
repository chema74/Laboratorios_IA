# Evidencias V2.1

Este directorio centraliza evidencias de la demo LLM opcional con Groq y fallback local.

## Archivos clave

- `demo_llm_groq.md`: resumen legible de la ejecución rápida.
- `resultado_demo_llm_groq.json`: salida estructurada con trazabilidad.
- `panel_demo.html`: dashboard estático para revisión.
- `interactivas/`: evidencias por ejecución del servidor interactivo.

## Notas

- No se guardan claves.
- Si Groq no está disponible, se usa fallback local determinista.
