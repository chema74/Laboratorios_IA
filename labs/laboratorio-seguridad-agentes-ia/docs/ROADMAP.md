# Roadmap

## V0 - Base documental
- Definición de alcance defensivo.
- Arquitectura conceptual local.
- Catálogo inicial de proyectos.
- Plantillas de riesgo e informe.

## V1 - Local funcional (objetivo posterior)
- Implementaciones locales mínimas y controladas.
- Trazabilidad básica de decisiones defensivas.
- Evaluación de políticas con escenarios sintéticos.

## V2 - Evaluación opcional de modelos
- Comparativas opcionales con APIs gratuitas cuando proceda.
- Fallback local para mantener reproducibilidad.
- Ampliación de métricas defensivas y evidencia.

## V3 - Integración empresarial defensiva (futuro)
- Integración con flujos internos de gestión de riesgo.
- Estandarización de reporting defensivo para operación.
- Gobernanza multiagente con controles por rol.

## Fuera de alcance actual
- Ataque real o explotación.
- Uso de malware o phishing.
- Producción con datos reales.
- Dependencia obligatoria de cloud o servicios de pago.
