# Enfoque de Seguridad Defensiva

## Alcance del repositorio
Este repositorio está dedicado a evaluación, prevención, detección y documentación defensiva de riesgos en agentes de IA.

## Qué sí hace
- Estructura riesgos de forma trazable.
- Simula entradas maliciosas ficticias en entorno seguro.
- Define límites y políticas de uso.
- Documenta evidencias y decisiones defensivas.

## Qué no hace
- No enseña ataque real.
- No desarrolla explotación.
- No incluye malware ni phishing real.
- No genera instrucciones ofensivas.

## Criterio metodológico
Cada artefacto debe poder revisarse, reproducirse localmente y auditarse sin dependencia de servicios externos ni uso de datos reales.
