# Decisiones Técnicas V1

## 1. Clasificación heurística local

La clasificación de riesgo se implementa con reglas heurísticas deterministas para asegurar reproducibilidad y trazabilidad técnica del resultado.

## 2. Sin APIs externas

No se integran APIs externas para evitar dependencia de proveedores, variabilidad de servicio y exposición de datos fuera del entorno local.

## 3. Datos sintéticos

Se usan datos sintéticos para permitir demostración pública, preservar privacidad y repetir escenarios de forma controlada.

## 4. Valor del inventario, evidencias y shadow IA

- El inventario estructura el universo de casos de uso IA.
- El registro de evidencias facilita auditoría documental interna.
- El análisis de shadow IA permite detectar iniciativas no formalizadas.

## 5. Valor de las fichas de sistema IA

Las fichas estandarizan documentación mínima por sistema y facilitan revisión organizativa antes de escalar uso.

## 6. Límites frente a auditoría legal o regulatoria real

Este laboratorio no emite dictámenes jurídicos, no certifica cumplimiento del AI Act y no sustituye evaluación legal, técnica y organizativa realizada por equipos especializados.

## 7. Enfoque de portfolio técnico

La V1 prioriza portabilidad, coste cero operativo y evidencia reproducible para fines formativos y demostrativos.
