# Catálogo de Proyectos Internos

1. **01-mapa-ecosistema-google-ia**: V1 local funcional mínima para mapear componentes Google IA sintéticos por categoría y capa.
2. **02-simulador-gmail-ia-empresarial**: V1 local funcional mínima para simular flujo de correo empresarial con clasificación y respuestas sintéticas.
3. **03-simulador-drive-docs-ia**: V1 local funcional mínima para simular gestión documental empresarial con resúmenes y etiquetas sintéticas.
4. **04-simulador-sheets-ia-analitica**: V1 local funcional mínima para simular analítica operativa con CSV sintético e indicadores.
5. **05-simulador-calendar-tareas-ia**: V1 local funcional mínima para simular agenda y tareas empresariales con reglas sintéticas.
6. **06-conector-gemini-fallback-local**: V1 local funcional mínima para fallback local determinista sin Gemini real.
7. **07-gobierno-permisos-google-simulado**: V1 local funcional mínima para evaluar permisos simulados con decisiones de gobierno defensivo.
8. **08-trazabilidad-automatizaciones-google**: V1 local funcional mínima para trazar automatizaciones simuladas con hash y evidencias.
9. **09-informe-google-ia-empresarial**: V1 local funcional mínima para consolidar resultados y estimar madurez simulada.
10. **10-demo-google-ia-empresarial-completa**: V1 local funcional mínima para demostrar cadena completa del laboratorio.
