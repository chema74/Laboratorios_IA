# Roadmap por Fases

## V0 Documental
- estructura base del repositorio;
- narrativa técnica, límites y alcance;
- catálogo y plantillas iniciales.

## V1 Local Funcional
- simuladores locales reproducibles;
- trazabilidad de flujo y evaluación de respuestas;
- gobierno de permisos ficticio.

## V2 Opcional con Gemini API
- integración opcional mediante `.env`;
- fallback local obligatorio en caso de ausencia de clave o servicio.

## V3 Futura (Extensión)
- posible integración con Google Workspace real;
- posible uso de Google Cloud como ampliación no obligatoria.

## Fuera de alcance actual
- producción real con clientes;
- datos reales o sensibles;
- dependencia obligatoria de servicios de pago;
- promesas de rendimiento/operación empresarial real.
