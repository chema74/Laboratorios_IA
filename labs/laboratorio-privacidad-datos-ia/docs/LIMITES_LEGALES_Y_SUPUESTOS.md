# Límites Legales y Supuestos

Este laboratorio tiene carácter **técnico, orientativo y de portfolio**.

## Alcance legal

- No emite dictámenes legales.
- No constituye asesoramiento jurídico.
- No garantiza cumplimiento RGPD.

## Naturaleza de las comprobaciones

- Las comprobaciones son técnicas, heurísticas y orientativas.
- La detección de PII no es exhaustiva.
- Las recomendaciones no sustituyen una evaluación regulatoria formal.

## Supuestos de ejecución

- Datos sintéticos con PII ficticia.
- Ejecución local sin cloud ni APIs externas.
- Evidencias documentales para revisión técnica.

## Requisito para uso real

Cualquier aplicación en producción requeriría revisión conjunta:
- jurídica,
- organizativa,
- técnica,
con especialistas de privacidad y cumplimiento.
