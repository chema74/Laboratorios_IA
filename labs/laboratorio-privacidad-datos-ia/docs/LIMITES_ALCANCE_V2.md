# Límites de alcance V2 — Laboratorio privacidad datos IA

## 1. Límite principal

Este laboratorio es una demostración técnica de privacidad aplicada a IA (Artificial Intelligence – Inteligencia Artificial), no una asesoría legal.

El repositorio puede ayudar a explicar buenas prácticas, riesgos habituales y artefactos de revisión, pero no determina cumplimiento normativo real.

## 2. Datos personales

La V2 no trabaja con datos personales reales.

Todo ejemplo debe interpretarse como sintético, simulado, anonimizado o de demostración.

Marcador explícito:

SIN_DATOS_PERSONALES_REALES: SI

## 3. RGPD y DPIA

Las referencias a RGPD (General Data Protection Regulation – Reglamento General de Protección de Datos) y DPIA (Data Protection Impact Assessment – Evaluación de Impacto relativa a la Protección de Datos) son referencias de contexto.

No equivalen a una evaluación legal formal.

## 4. AI Act

Las referencias al AI Act (Artificial Intelligence Act – Reglamento de Inteligencia Artificial) se usan como contexto de gobernanza de IA.

No implican clasificación jurídica definitiva de sistemas ni dictamen normativo.

## 5. Tecnología

La V2 se mantiene local-first.

No requiere:

- Cloud obligatorio.
- Claves reales.
- Tarjeta bancaria.
- APIs (Application Programming Interfaces – Interfaces de Programación de Aplicaciones) de pago.
- Servicios externos cerrados.

## 6. Uso profesional correcto

Este laboratorio se puede usar como evidencia de portfolio técnico para mostrar criterio, trazabilidad y diseño responsable.

No debe venderse como producto legal, certificación ni auditoría oficial.
