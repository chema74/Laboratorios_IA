# Límites de alcance V2 — Laboratorio backend IA empresarial

## 1. Límite principal

Este laboratorio es una demostración técnica de backend IA empresarial local-first, no una plataforma productiva cerrada.

Su función es mostrar arquitectura, trazabilidad, validación y generación de evidencias, no operar como infraestructura empresarial real.

## 2. Infraestructura

La V2 no requiere:

- Servidores cloud.
- Kubernetes.
- Docker obligatorio.
- Bases de datos gestionadas.
- Colas empresariales externas.
- Pasarelas comerciales.
- Sistemas de autenticación productiva.

Todo debe poder revisarse en local.

## 3. Modelos y APIs

La V2 no obliga a usar APIs (Application Programming Interfaces – Interfaces de Programación de Aplicaciones) de pago ni claves reales.

Las integraciones con proveedores externos quedan como posibilidad futura documentable mediante `.env` y siempre con fallback local.

## 4. Seguridad

La V2 puede documentar buenas prácticas de seguridad, pero no sustituye una auditoría formal de ciberseguridad, pentesting, revisión de cumplimiento, hardening productivo ni análisis legal.

## 5. Producción

La V2 no promete disponibilidad, escalabilidad, tolerancia a fallos ni métricas productivas.

Representa una base de portfolio revisable y ampliable.
